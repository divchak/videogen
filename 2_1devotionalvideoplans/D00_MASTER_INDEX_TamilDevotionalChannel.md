## 🏗️ PRODUCTION WORKFLOW — STEP BY STEP

### For Each Song:

#### STEP 1: AUDIO (Suno AI)
1. Use the Suno AI prompt from the song's plan file
2. Generate 3–4 variations
3. Select the one with the most authentic Carnatic feel
4. Download at maximum quality
5. Clean up any artifacts in Audacity or similar

#### STEP 2: LYRICS/TEXT OVERLAYS
1. Prepare Tamil lyrics text
2. Prepare English transliteration
3. Prepare English meaning (for diaspora audience)
4. Format as .srt subtitle file with 10-second timing blocks

#### STEP 3: CHARACTER IMAGES (Image AI)
1. Use the character image prompts from each plan file
2. Generate each character in the following views:
   - Front facing (main)
   - Side profile
   - 3/4 view
   - Key emotional moment
3. Tools: Midjourney / DALL-E 3 / Ideogram (use "Pixar 3D sacred" in all)

#### STEP 4: SCENE GENERATION (Video AI or Image sequence)
1. Use each scene prompt from the plan file
2. Generate scene images (1 per 10-second block)
3. Apply slow motion zoom/pan (Ken Burns effect) to each image
4. Tools: Kling AI / Runway / Sora for animation, or image-to-video

#### STEP 5: VIDEO ASSEMBLY
1. Import all scene images/clips into editing software
2. Sync to the Suno AI audio track
3. Add subtitle overlays (Tamil + English)
4. Add scene transitions (cross-dissolve, 0.5 sec)
5. Add opening title card (3 seconds)
6. Add closing title card with channel branding (5 seconds)
7. Export at 4K for maximum YouTube quality

#### STEP 6: UPLOAD OPTIMIZATION
1. Title: "[Tamil Song Name] 3D Animation | [Deity Name] | Tamil Devotional"
2. Thumbnail: Key deity close-up + golden glow + large Tamil text
3. Description: Full Tamil lyrics + English meaning + all deities tagged
4. Tags: Tamil script name + Roman transliteration + festival name + deity name
5. Upload timing: Per the schedule above, 2-3 weeks before the festival peak
